Console Game

A console based pokemon game written using C++ is an implementation of Object Oriented Programming.
Game include some worth mentioning features such as type advantages of certain types of pokemon over other types of pokemon, i.e. Fire type being stronger than Grass type etc causing them to deal more damage and taking less damage. There are other features too such as decreasing confidence of pokemon on continuous escapes during battle resulting in a slight decrease in damage inflicted to enemy.

I am thinking to work on adding more pokemons and a database( A txt file based base ) to store User's progress and creating a logging in system, but for the time being, it was just a starter project.

Project link: Pokemon
Tags: C++, OOPs
